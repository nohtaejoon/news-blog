# 파이썬 vs 자바스크립트 - 初学者 선택 가이드

## 결론

**목표에 따라 다르다.**
백엔드/데이터 → 파이썬, 프론트엔드/웹 → 자바스크립트

---

## 1. 언어 비교

| 기준 | 파이썬 | 자바스크립트 |
|------|-------|-------------|
| 문법 | Easy | 중간 |
| 활용처 | 데이터, AI, 백엔드 | 웹, 프론트엔드 |
| 커뮤니티 | 크고 활발 | 크고 활발 |
|_jobs | 많음 | 많음 |
| 학습 곡선 | 완만한 | 중간 |

## 2. 추천 학습 로드맵

### A. 웹 개발자:
```
1. HTML/CSS 기초 → 1주
2. 자바스크립트 → 4주
3. React/Next.js → 4주
4. 파이썬 (선택) → 2주
```

### B. 데이터/AI 개발자:
```
1. 파이썬 기초 → 4주
2. pandas/NumPy → 4주
3. 딥러닝 → 8주
4. 자바스크립트 (선택) → 2주
```

## 3. 결론

> **"둘 다 배울 수 있다면 이상적이지만,**
> **먼저 하나에 집중하라."**
