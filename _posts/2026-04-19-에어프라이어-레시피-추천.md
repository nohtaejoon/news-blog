# 에어프라이어 추천 레시피 - 초간편 요리

## 결론

**에어프라이어=초간편+건강+맛있음.**
매일쓰게 되는 주방 도구.

---

## 1. 추천 레시피 5選

### 1. 감자 튀김 (20분)
```
감자 슬라이스 → 올리브오일 발라 →
180도 20분 → 소금
```

### 2. 닭다리 구이 (25분)
```
닭다리 소금/후추 → 200도 25분 →
바삭함
```

### 3. 연어 구이 (12분)
```
연어 소금/레몬 → 200도 12분 →
풍미满자
```

### 4. 채소 튀김 (15분)
```
브로콜리, 당근 → 올리브오일 →
180도 15분
```

### 5. 새우 튀김 (10분)
```
새우 소금/后추 → 가루 →
180도 10분
```

## 2. 핵심 팁

- 온도: 180-200도
- 시간: thick 따라 조정
- 오일: 올리브오일 추천

---

**에어프라이어=시간 아끼고 맛있고 건강한 한식.**
